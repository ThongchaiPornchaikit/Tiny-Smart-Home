# Tiny-Smart-Home
Physical Computing Project 2025 - IT KMITL


วัตถุประสงค์และฟังก์ชันการทำงาน (Objectives and Functionality)

โครงงานนี้มีวัตถุประสงค์เพื่อสร้างระบบบ้านอัจฉริยะที่มีฟังก์ชันการทำงานอัตโนมัติ 3 ส่วนหลัก ดังนี้:

1. ระบบประตูอัตโนมัติ
* การตรวจจับ:ระบบจะใช้เซ็นเซอร์ตรวจจับระยะห่างของคนกับประตู
* การทำงาน:เมื่อมีคนเข้ามาใกล้ในระยะที่กำหนด ประตูจะเปิดให้โดยอัตโนมัติและจะปิดเองหลังจากเวลาผ่านไประยะหนึ่ง
* การควบคุมเพิ่มเติม:ผู้ใช้สามารถปรับโหมดการทำงานเป็นแบบ MANUAL (ควบคุมด้วยตนเอง) ได้ผ่านทางหน้า WEBSITE

2. ระบบไฟส่องสว่างอัตโนมัติ
* การตรวจจับ:ระบบจะตรวจจับระดับแสงสว่างภายในบ้าน
* การทำงาน:เมื่อแสงในบ้านน้อยจนถึงจุดที่กำหนด หลอดไฟจะสว่างขึ้นอัตโนมัติ และในทางกลับกัน หากมีแสงสว่างเพียงพอ หลอดไฟก็จะปิดเอง
* การควบคุมเพิ่มเติม:ผู้ใช้สามารถสั่งเปลี่ยนสีไฟ หรือสั่งปิดไฟ ได้ผ่านทางหน้า WEBSITE

3. ระบบแสดงผล
* การแจ้งสถานะ:มีจอ LCD ติดตั้งอยู่ด้านข้างประตู เพื่อแสดงรายละเอียดสถานะเมื่อประตูเปิดหรือปิด
* การต้อนรับ:จอ LCD จะแสดงข้อความ "WELCOME"


สรุปหลักการทำงานและอุปกรณ์ (Summary of Working Principles and Components)

เพื่อให้บรรลุวัตถุประสงค์ข้างต้น โครงงานนี้ใช้หลักการทำงานและส่วนประกอบหลักดังนี้:

ตัวควบคุมหลัก (Main Controller):
    * ARDUINO UNO R4 WIFI:เป็นหน่วยประมวลผลกลาง ทำหน้าที่สั่งการให้ Component อื่นๆ ทำงานตามที่โค้ดได้กำหนดไว้

ส่วนการทำงานของประตู (Door Mechanism):
    * ULTRASONIC SENSOR: ทำหน้าที่รับข้อมูล (ตรวจจับระยะห่างของวัตถุหรือคน)
    * MICRO SERVO MOTOR: ทำหน้าที่หมุนเพื่อเปิดและปิดประตู ตามคำสั่งที่ได้รับจาก Arduino.

ส่วนการทำงานของไฟ (Lighting Mechanism):
    * LDR SENSOR: ทำหน้าที่ตรวจจับความสว่างของแสงโดยรอบ เมื่อเซ็นเซอร์ตรวจจับได้ว่าแสงน้อย ก็จะส่งสัญญาณไปให้ Arduino เพื่อสั่งเปิดไฟ
